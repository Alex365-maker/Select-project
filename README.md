## Running

Open VSCode
- Terminal -> Create Terminal
- In the created terminal, you need to write:
```
git clone https://github.com/Alex365-maker/Select-project.git
cd Select-project
yarn
yarn dev
```
- Open the link in the browser http://localhost:3000/
